# 🎮 My-Metroidvania: 基于有限状态机的 2D 动作游戏框架

## 📌 项目定位
本项目不仅是一个《类银河恶魔城》游戏 Demo，更是一套针对 **2D 动作游戏** 封装的通用底层架构。开发重点在于解决复杂状态切换下的代码解耦与物理系统的精密控制。

## 🛠️ 技术深度 (面试亮点)
- **解耦式状态机 (FSM)**：采用接口隔离原则，实现玩家与敌人行为逻辑的高度可复用，支持动态状态注入。
- **事件驱动反馈系统**：基于 C# 委托（Action）实现攻击检测与受击反馈，确保战斗模块与其他系统（如 UI、音效）零耦合。
- **精密 2D 控制器**：自研射线检测（Raycast）碰撞方案，完美解决 Unity 物理引擎在高速移动下的“卡墙”与“穿模”痛点。
- **高性能视差系统**：通过背景层坐标映射算法，在维持 60FPS 的前提下提供极佳的纵深表现力。
